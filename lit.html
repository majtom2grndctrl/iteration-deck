Component Features

Demonstrating the full feature set of iteration deck components

Features Showcase

  • Multi-deck Support: Multiple decks on the same page with shared toolbar
  • Keyboard Navigation: Use Ctrl/Cmd + Arrow keys to navigate slides
  • Development Toolbar: Fixed position toolbar with deck selector dropdown
  • Production Mode: Renders only first slide in production builds
  • AI Context: Support for AI prompts and generation metadata
  • Universal Compatibility: Works with any framework or vanilla HTML

Usage Instructions

  1. Import the web components module: import '/src/web-components/index.ts'
  2. Use kebab-case element names: <iteration-deck> and <iteration-deck-slide>
  3. Provide unique id for each deck and descriptive label for slides
  4. The toolbar will automatically appear in development mode
  5. Use keyboard shortcuts to navigate between slides in the active deck
